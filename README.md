# grabIt

**grabIt** is a rental marketplace web application built with **Django (backend)** and **React (frontend)**.  
It allows users to list, search, and book properties or vehicles for rent. The project is inspired by platforms like **OLX, Revv Cars, and 99acres**, with additional features such as chat, bookings, and payments.

---

## 🚀 Features
- 🔐 User authentication (Sign up, Login, Logout)
- 🏠 List rental properties and vehicles
- 🔍 Search and filter listings
- 📅 Booking system (daily rental support)
- 💬 Real-time chat between owners and customers (Django Channels + WebSockets)
- 💳 Secure payments integration with Cashfree
- 📧 Email notifications for bookings
- 📊 User dashboard to manage listings, bookings, and chats

---

## 🛠️ Tech Stack
**Frontend:** React 19, Tailwind CSS (planned)  
**Backend:** Django / Django REST Framework  
**Database:** PostgreSQL  
**Real-time:** Django Channels, WebSockets  
**Payment Gateway:** Cashfree  
**Version Control:** Git & GitHub
